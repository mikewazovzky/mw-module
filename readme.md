Just a Stupid Logger 
=========

A test library that logs out argument string

## Installation

  `npm install mw-module-test`

## Usage

    var logger = require('mw-module-test`');

    var str = logger('Test string');
  
  
  Output should be `Test string`


## Tests

  `npm test`
